# Multi-bracket Validation

* Write a function called `validate_brackets`
  * Arguments: string
  * Return: boolean representing whether or not the brackets in the string are balanced
* Types of brackets:
  * Round Brackets : ()
  * Square Brackets : []
  * Curly Brackets : {}

## Approach & Efficiency

**BigO**

* Space: O(n) Because we are adding to the stack.
